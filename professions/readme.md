# WoW Midnight — Profession Guide

> Een all-in-one offline/PWA companion app voor alle 13 professions in World of Warcraft: Midnight.

🔗 **Live:** [huijting.github.io/midnight_guide/professions/](https://huijting.github.io/midnight_guide/professions/)  
🏠 **Hoofdapp (Dungeons):** [huijting.github.io/midnight_guide/](https://huijting.github.io/midnight_guide/)

---

## Inhoud

De app bevat voor **alle 13 professions** (8 Crafting + 3 Gathering + 2 Secundair):

| Tab | Inhoud |
|-----|--------|
| 📋 Overzicht | Tier, goud-rating, nut, lore |
| 🌳 Specialisaties | Stap-voor-stap build met point-aantallen |
| ⭐ Beste Items | Wat te craften + categorie-tags |
| 📍 Trainer | Naam, locatie, `/way`-coördinaten |
| 📦 Crafting Orders | Hoe werkt het + tips |

## GitHub structuur

```
midnight_guide/
├── index.html              ← Dungeon Guide (hoofd-app)
├── professions/
│   └── index.html          ← Profession Guide (deze app)
└── README.md
```

Aanbevolen: zet de profession guide in een `professions/` subfolder van het bestaande `midnight_guide` repo. Zo gebruik je de al-ingestelde GitHub Pages zonder nieuwe repo.

---

## Installatie

1. Kopieer `professions_guide.html` naar je repo als `professions/index.html`
2. Commit & push naar `main`
3. GitHub Pages serveert automatisch op `huijting.github.io/midnight_guide/professions/`
